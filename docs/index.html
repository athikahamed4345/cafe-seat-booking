Features

👥

For Customers

Register securely, browse available seats by zone and time, book instantly, and manage all your reservations from one dashboard.

⚙️

For Admins

View all bookings, manage reservations, control table availability, and track live statistics in a powerful admin dashboard.

🔒

Secure & Compliant

JWT authentication, BCrypt encryption, GDPR compliance, email verification, and role-based access control built in.

Technology Stack

Backend

Java 21, Spring Boot 3.2, Spring Security, Spring Data JPA with comprehensive REST APIs.

Frontend

HTML5, CSS3, Vanilla JavaScript with Fetch API. Responsive design with no frameworks needed.

Database

H2 for local development, PostgreSQL for production deployment on Railway or Render.

Authentication

JWT tokens with 24-hour expiry, BCrypt password hashing, role-based access control.

Email System

Automated emails for verification, booking confirmations, status updates via Gmail or SendGrid.

Testing

15 JUnit 5 tests with Mockito covering booking and authentication services.
